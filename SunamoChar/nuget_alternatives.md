# NuGet Alternatives to SunamoChar

This document lists popular NuGet packages that provide similar functionality to SunamoChar.

## Overview

Character manipulation utilities

## Alternative Packages

### System.Char
- **NuGet**: System.Runtime
- **Purpose**: Built-in .NET character methods
- **Key Features**: IsDigit, IsLetter, ToUpper, ToLower, Unicode support

### Humanizer
- **NuGet**: Humanizer
- **Purpose**: String and character humanization
- **Key Features**: Pluralization, casing, number to words conversion

## Comparison Notes

Most character operations are well-covered by built-in System.Char methods.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Char**: Built-in .NET character methods
- **Humanizer**: String and character humanization
